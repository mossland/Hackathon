# 사용자 데이터 기반 Metaverse AI 아바타

### Brief Description  
사용자의 대화 데이터(메신저, SNS 기록 등)를 학습하여, 해당 사용자의 말투와 스타일을 재현하는 AI 아바타를 Mossland Metaverse에 구현

### Proposed Solution  
- 표시하려고 하는 사용자에 대한 기반 데이터 업로드 후 학습
  - 대화 로그 데이터
  - SNS 데이터 내역 등
- 신규 모델을 구축하거나 기존 LLM을 사용하여 자신이 답변할 메세지인지 판단하고 답변
- 귓속말 기능을 추가하여 좀 더 친밀하고 자연스러운 대화 가능하게 함
- 고인(지인, 과거의 위인 등)의 데이터를 기반으로 학습시켜 해당 인물과 대화하는 경험 제공

### Benefits for Mossland  
- 사용자가 감정적으로 깊은 연결을 느낄 수 있는 독특한 메타버스 경험을 제공
- 사용자 참여 및 체류 시간 증가
- 다양한 활용 분야(추억 회상, 역사 체험, 커뮤니티 활성화 등)로 확장 가능

### Possible Challenges  
- 개인정보 보호 및 데이터 수집·활용에 관한 법적, 윤리적 이슈
- AI 아바타의 답변 일관성과 자연스러움 확보
